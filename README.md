<h2>실행 방법</h2>
<ol>
<li> 패키지 매니저 yarn 설치<br>
https://classic.yarnpkg.com/en/docs/install#windows-stable</li> 
<li>프로젝트 폴더로 이동<br>
$cd dalgona-vote-front</li>
<li>패키지 설치<br>
$yarn</li>
<li>실행<br>
$yarn start</li>
</ol>
